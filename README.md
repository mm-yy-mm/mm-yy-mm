## 🔬 Experiments Repository (Separate From Portfolio)

### 🧭 Why This Lab Exists

This lab is not meant to impress anyone. It exists for one reason only:

**to learn by doing, breaking, and rebuilding.**

Some repositories aim to be perfect.
This one aims to be **honest**.

Inside this lab, you will find unfinished ideas, half-implemented concepts, rewrites, refactors, reversions, and experiments that go nowhere — and that is the point. Real learning is not a straight line.

If you are browsing here, take it as an invitation to:

- read broken code without fear
- question assumptions
- trace mistakes
- watch ideas evolve over time

No promises, no polish — just progress.
